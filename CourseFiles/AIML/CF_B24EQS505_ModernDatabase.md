# Course File: Modern database

## 0. Course Overview
This course will provide students with a general overview of NOSQL database, 
introducing the different modern database systems, Illustrating the features of 
NOSQL, develop the real time application using pymongo.  

## 1. Course Outcomes (COs)
- CO1 (Remember→Understand): Compare RDBMS and NoSQL databases.
- CO2 (Understand→Apply): Develop real-world applications using MongoDB and PyMongo.
- CO3 (Apply): Build document-oriented database applications using MongoDB.
- CO4 (Apply→Analyze): Apply MongoDB features for efficient data management.
- CO5 (Analyze→Evaluate): Choose suitable databases for different application scenarios.

## 2. Mapping of Units to COs

| Unit | Topics Covered | Mapped CO(s) |
|------|----------------|--------------|
| **Unit I** | Introduction to Modern Databases, RDBMS vs NoSQL, Database Selection Strategy | **CO1, CO5** |
| **Unit II** | MongoDB Overview, PyMongo, Data Modelling, Database & Collection Creation | **CO2, CO3** |
| **Unit III** | CRUD Operations, Querying, Projection, Indexing, Aggregation, Replication, Sharding, Backup | **CO3, CO4** |
| **Unit IV** | Mini Project: Design and implementation of a MongoDB-based real-world application using PyMongo | **CO2, CO3, CO4, CO5** |

## 3. Syllabus (Unit-wise)

 ### Unit 1: Introduction to Modern Databases (Weeks 1–4)

- Introduction to Modern Database, Overview, CAP Theorem and BASE Properties, Selecting the Optimal Database, Relational & Non-Relational 
Systems, Comparison of RDBMS & NOSQL DB, Popular Modern Databases, Other Database Options, Strategy to choose the Database 

### Unit 2: MongoDB with PyMongo (Weeks 5–7)
- Mongo DB with pymongo : MongoDB ─ Overview, Advantages, Environment, Data Modelling, Create Database, Drop Database, Create Collection, Drop Collection, Datatypes 

### Unit 3: MongoDB Operations & Features (Weeks 8–10)
- Insert Document, Query Document, Update Document, Delete Document, Projection, Limiting Records, Sorting Records, Indexing, Aggregation, Replication, Sharding, Create Backup 

### Unit 4: Mini Project (Weeks 11–15)
- Mini Project: Design and implementation of a MongoDB-based real-world application using PyMongo.
