## 1. Course description 🟢

This course provides a conceptual and foundational introduction to Artificial Intelligence (AI) and Machine Learning (ML) tailored for modern engineering disciplines. It covers the evolution of intelligent systems, Turing test constraints, and the agent-environment loop. Students explore classical search strategies (BFS, DFS, A*), propositional logic, and optimization models. The course introduces machine learning paradigms (supervised, unsupervised, reinforcement), regression/classification models, neural networks (Perceptrons), and model diagnostic boundaries (underfitting/overfitting). Additionally, students learn the fundamentals of Generative AI, Large Language Models (LLMs), prompt engineering basics, and discuss the ethical implications, biases, safety considerations, and engineering application cases of AI.

## 2. Course content (units & weightage) 🟢

| Unit | Syllabus | Weightage |
|---|---|---|
| 1 | **Introduction to AI & Intelligent Agents:** Defining AI, history and milestones, Turing Test. Intelligent Agents: Agent-Environment interaction loop, sensor-actuator structures, PEAS descriptors (Performance, Environment, Actuators, Sensors). Characterization of environments (fully/partially observable, deterministic/stochastic, static/dynamic, discrete/continuous). Simple reflex agents and goal-based agent architectures. | 12.5 Marks |
| 2 | **Search, Logic & Optimization:** Problem-solving as state-space search. Uninformed search strategies: Breadth-First Search (BFS), Depth-First Search (DFS); time and space complexity bounds. Informed (Heuristic) search: A* search algorithm, heuristic admissibility and consistency. Basics of propositional logic, inference rules, and constraint satisfaction. Optimization search concepts: Hill Climbing and local extrema. | 12.5 Marks |
| 3 | **Machine Learning & Neural Foundations:** Machine Learning paradigms: Supervised learning (classification, regression), Unsupervised learning (clustering), and Reinforcement learning (agent reward loops). Basic ML algorithms: Linear Regression, Decision Trees. Neural Networks: Artificial neurons, activation functions (Sigmoid, ReLU), Single-layer Perceptrons, and Multi-Layer Perceptron (MLP) structures. Model diagnostics: Underfitting, Overfitting, training/test splits. | 12.5 Marks |
| 4 | **Modern Generative AI & Societal Impact:** Foundations of Generative AI: Transformers, Large Language Models (LLMs), and tokenization. Practical Prompt Engineering: techniques (Zero-shot, Few-shot), prompt constraints, and AI-assisted development tools (GitHub Copilot). Engineering application cases. AI Ethics and Responsibility: algorithmic bias, data privacy, safety boundaries, transparency, and the future of engineering roles. | 12.5 Marks |

## 6. Pedagogy 🟢

This course is delivered using classroom lectures combined with interactive concept-checking activities, following a theory-based credit structure (1-0-0-1).
- **Interactive Lectures (1 hour/week):** Focus on delivering core conceptual principles, algorithm demonstrations, and visual illustrations of agent-environments.
- **Active Learning Breaks:** Every lecture incorporates short formative checkpoints (concept quizzes, peer discussions, and prompt-writing activities) to verify student comprehension in real-time.
- **Assessment Model (Theory-only ETC):** The course follows the **IA1 + IA2 + SEE** assessment type tailored for 1-credit theory courses. Continuous evaluation (IA1, IA2, and Assignments) is combined with a Semester End Examination (SEE) to measure learning outcomes.

## 8. 🔵 Differentiated instruction (mapped to the two levels)

**Reaching the floor (awareness level) — for students at risk of not clearing it:**
- **Visual Mapping Worksheets:** Provide structured hand-out templates for drawing state-space search trees (BFS/DFS) and manual A* calculations.
- **Formative Diagnostics:** Deliver weekly online practice quizzes on the LMS to target common conceptual misconceptions (e.g. confusing supervised vs unsupervised learning).
- **AI Tool Templates:** Provide pre-constructed system instructions and prompts so students can practice variables changes without starting from scratch.
- **Remediation Peer-Mentoring:** Partner students struggling with basic perceptron arithmetic with peer mentors during review blocks.

**Reaching the ceiling (advanced level) — for students aiming above 8 CGPA:**
- *Note:* Since this is a 1-credit Level A (Awareness only) course, formal advanced assessments are N/A. However, high achievers are guided towards early enrichment:
  - **Self-Directed LLM APIs Exploration:** Guide students to use free API tiers to orchestrate multi-step prompt workflows.
  - **Frontier Research Reading:** Provide optional short reading lists on AI safety, model alignment (RLHF), and agent loops.
  - **Future SIG Track Advising:** Map their performance directly to advise selections in the AI/ML SIG basket starting in Semester 3.

## 22. 🔵 Implementation strategy — Merrill's First Principles

This course maps session transactions to **Merrill's First Principles of Instruction**, reinforcing systematic cognitive progression.

- **Problem / Task-centred:** Lectures begin by introducing an anchoring engineering challenge (e.g., path navigation, classification categorization, automatic response structures).
- **Activation:** Existing knowledge of basic functions, probability, and programming logic is activated using concept recalls and recap drills.
- **Demonstration:** Algorithms (BFS/DFS, A*) and neural structures (Perceptrons) are demonstrated by the instructor using visualizations and step-by-step math traces.
- **Application:** Students apply their knowledge in short concept checks, trace searches manually, and test prompt engineering structures.
- **Integration:** Students compile project profiles and prompt engineering portfolios to solve simulated engineering support tasks.
