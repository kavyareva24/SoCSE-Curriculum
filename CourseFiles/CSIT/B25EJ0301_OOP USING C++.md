# Course Overview

This course introduces students to the principles of Object-Oriented Programming (OOP) using C++. Students learn to design and implement classes, apply encapsulation, inheritance, and polymorphism, and develop robust applications using templates, exception handling, and file handling. The course also introduces modern C++ (C++11/14/17) features such as move semantics, smart pointers, lambda expressions, and the Standard Template Library (STL), enabling students to build efficient and industry-ready software solutions.

---

# Course Objectives

Upon successful completion of this course, students will be able to:

1. Explain the fundamental concepts of Object-Oriented Programming, including encapsulation, abstraction, inheritance, and polymorphism.
2. Implement user-defined data types using constructors, operator overloading, copy constructors, and friend functions.
3. Design applications using inheritance, virtual functions, and runtime polymorphism.
4. Develop robust C++ programs using templates, exception handling, and file input/output.
5. Apply modern C++11/14/17 features including move semantics, smart pointers, lambda expressions, and STL containers.
6. Evaluate and implement best programming practices including RAII, const correctness, and memory-safe programming.
