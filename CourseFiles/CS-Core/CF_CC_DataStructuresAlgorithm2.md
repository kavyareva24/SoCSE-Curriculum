# Course File — Advanced Data Structures and Algorithms

> **Status: DRAFT** — This course is redesigned for third-year B.Tech students as a practical, placement-oriented DSA course. The syllabus is limited to topics that are feasible within the academic level and are directly useful for coding interviews, competitive programming, and engineering problem solving.

---

## 0. Course Overview 🟢

This course strengthens the problem-solving ability of third-year B.Tech students through advanced data structures and algorithmic thinking. It focuses on topics that are commonly asked in internships and placement drives, including balanced trees, heaps, graph algorithms, greedy techniques, dynamic programming, and string algorithms. The emphasis is on implementation, correctness, complexity analysis, and practical problem solving rather than excessive theoretical depth.

---

## 1. Course Outcomes (COs) 🟢

- CO1 (Understand→Apply): Explain and implement advanced tree structures such as AVL trees, Red-Black trees, heaps, tries, segment trees, and Fenwick trees for solving standard problems.
- CO2 (Apply→Analyze): Represent graphs and apply BFS, DFS, topological sorting, cycle detection, union-find, MST, and shortest path algorithms to solve real-world and interview-style problems.
- CO3 (Apply→Analyze): Apply greedy strategy to solve standard optimization problems such as activity selection, knapsack, Huffman coding, scheduling, and merging problems.
- CO4 (Analyze→Evaluate): Design and compare dynamic programming and backtracking solutions for problems such as knapsack, LCS, LIS, edit distance, and N-Queens.
- CO5 (Evaluate→Create): Develop efficient, testable, and well-documented solutions for placement-oriented problems in trees, graphs, greedy methods, dynamic programming, and string algorithms.

## 3. Course Design Approach 🟢

The course is designed to be doable for third-year students by focusing on core implementation techniques and commonly asked interview problems. Advanced topics such as AVL and Red-Black trees, shortest path algorithms, and trie-based pattern problems are introduced at an application-oriented level rather than a deep theoretical level.

---
