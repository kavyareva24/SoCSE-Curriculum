# **4\. Course Objectives**

The objectives of this course are to enable students to:

- Understand the fundamental concepts, types, and workflow of machine learning systems.
- Apply data preprocessing, feature engineering, and data preparation techniques for machine learning tasks.
- Develop regression models for predictive analysis and performance evaluation.
- Build and evaluate classification models using various machine learning algorithms.
- Analyze and improve machine learning models using validation, tuning, and ensemble techniques.
- Apply clustering and dimensionality reduction methods to extract meaningful insights from data while understanding responsible AI principles.

# **9\. Pedagogical Framework - Merrill's First Principles of Instruction**

The plan is built around Merrill's five instructional phases, cycling within each unit:

- Activation - quizzes that surface prior knowledge before a new sub-topic (used 12 times, mostly Awareness-level).
- Demonstration - worked examples and guided demonstrations that model the skill (13 sessions).
- Application - programming exercises, Role plays, case studies and scenario-based problem solving where students apply machine learning concepts.
- Integration - recap and comparative sessions that consolidate a unit and connect it to the next (4 sessions, one per unit).
- Problem-Centred - a motivating, real-world problem opens Units II, III and IV to frame why the unit's technique is needed (3 sessions).

# **13\. CO Attainment Calculation Methodology**

Since each CO carries an Awareness and (where applicable) an Advanced component, attainment is recommended to be computed as a weighted blend:

- Direct attainment = 0.4 × (average score on Awareness-tagged items) + 0.6 × (average score on Advanced-tagged items), computed per CO across CIE-I/II/III, continuous assessment, and SEE.
- Indirect attainment = course-end survey / student feedback on the same CO statements (typically weighted 20% against 80% direct attainment).
- Target threshold: typically 60% of students scoring ≥ 60% marks on the CO's associated questions (institution-specific threshold to be confirmed).
